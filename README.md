## 🧠 What is UrbanPulse?
**UrbanPulse** is an AI-powered architectural decision simulator that generates explicit real-time models of how civic policies impact traffic, economic health, ecology, and citizen sentiment.

Instead of guessing outcomes, policymakers can simulate, compare, and instantly understand hyper-local decisions—from removing parking spaces to building metro lines—before breaking ground. 

## ⚡ Key Features
* **Neural Feedback Engine:** Watch virtual citizens and businesses fiercely debate your proposed policies in real-time streams based directly on the economic logic.
* **Temporal Simulation:** Slide the temporal bounds from 1 to 10 years to extrapolate compounding structural outcomes over time.
* **Variant Comparison:** Input strategic directives including budget limits, risk tolerance, and optimization targets to explicitly pit Policy A vs Policy B and force the AI to recommend a mathematical winner.
* **Macro Analytics Dashboard:** Immediate visual feedback powered by Recharts radar vectors highlighting precise algorithmic deviations across the city's health score.

## 🛠️ Architecture
This repository contains the **Spring Boot (Java)** backend API module. 
* **Backend:** Spring Boot, RestTemplate, Jackson
* **Frontend:** React, Tailwind CSS, Vite (hosted in the [UrbanPulse-Frontend](https://github.com/Akshatsharma2205/UrbanPulse-Frontend) repository)
* **AI Model:** Google Gemini 2.5 Flash API Configuration
